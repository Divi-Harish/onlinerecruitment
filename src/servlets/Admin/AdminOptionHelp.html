Admin Option Help!

This page is Activated when admin clicks the Admin Option in the Admin tool page.

In Admin! JSP page Admin can Create a new Admin Profile inputting all the details as required. A Admin Profile is needed to log on to the application for performing the operations meant for Administration.
Admin can Edit an existing AdminProfile or s/he can Delete it either. The list of all available Profiles are displayed for the Admin to select and to perform the operations.
