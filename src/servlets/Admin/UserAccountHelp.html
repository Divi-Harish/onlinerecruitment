User Updation Help!

This page is Activated when admin clicks the User Update! in the Admin tool page.

Admin can here Create a new User, s/he can View the list of all the existing users. Admin can also Modify the User Profile. Admin has the rights to revoke some existing user.

When the "Edit User" Button is clicked in the User Updation Page a JSP page is called with all the values of the user selected from the list of the users. Admin can then modify (or leave the values as it is) and can Update the User Profile in the DataBase.
When the "Delete User" Button is clicked in the User Updation Page a Servlet is called which deletes the specified user from the database. A Warning is given before deleting the selected user.
