Introduction
Thank you for installing this release of Online Recruitment SystemTM Standard Edition Version 1.5 . The System provides a perfect environment for carrying out all the operations related to  recruitment for the companies.

The system includes various tools which can be used to dynamically create tests, calculate results and analyse the performance of the candidates.

System Requirements & Installation

System requirements :

  • Windows (NT,2000,XP) based Operating System.
  • Oracle 8i or 9i Enterprise/Standard Edition.
  • Java Virtual Machine 1.5.0 (Windows Platform).
  • Apache Tomcat 4.0.1. OR Apache Tomcat 4.1.18-LE.

Installation Instructions :

  • Install Java Virtual Machine 1.5.0 (Windows Platform).
  • Install Apache Tomcat 4.0.1. OR Apache Tomcat 4.1.18-LE.
  • Install Oracle 8i or 9i Enterprise/Standard Edition.
  • Install Online Recruitment System Ver 1.1 by invoking the file "Setup.bat".
  • After installation start Tomcat from the path given in the Readme and type the respective URL to run  the Administrator or the User part.

Troubleshooting :

  • During Oracle 8i installation, while creating the Database if the following error occurs : 'ORA-12560: TNS:protocol adapter error'  then 
    1.  Choose "Abort".
    2.  Select "Oracle Database Configuration Assistant" and click "Retry".
    3.  Select "Yes" in order to delete the files to setup the database.
  • If an error occurs "Oracle not available", then select the file "Startup.bat" in the console type "startup" and press Enter. type "exit" to close the program.
  • If any other exception occurs during installation then
    1.  Check the system requirements of the software.
    2. Check whether all the software required are installed and running.
    3. Check if database for oracle is created.
    4. If the setup file "Setup.jar" is corrupt or damaged then try downloading new file.
    5. On performing all these checkups, if the problem persists then do inform us at www.pranav_558@yahoo.com.

Release Notes

This System is restricted to only Windows Platform. The System is a web based application and runs on Apache Tomcat WebServer. Number of machines supported by the System is currently unknown. The System presently is restricted to LAN and does not have much inbuilt security features.

Compatibility

The new version 1.5 is completely compatibile with the previous releases. The database of the new system is designed considering the pervious releases and can easily accommodate the database of the pervious releases. Also with the help of Oracle Co-operation we have been able to provide users the facilities to switch from Oracle 8i to Oracle 9i without loosing the present data. Also all other configuration and log files are compatile with this new version.
