Questions [ Help Index ]

The Questions in the Category form the vital part which distinguish one Category from another. The Questions of three different types form the test for the respective Category. The Questions are divided according to their format namely in three types :

1: Multiple Choice - The questions of this type have multiple options of which only one choice is the answer.

2: Multiple Answer - The questions of this type have multiple options with none or one or more than one answers correct.

3: True or False - The questions of this type have only two choices with one answer correct while the being the wrong answer.
