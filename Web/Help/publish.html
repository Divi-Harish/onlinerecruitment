Publish Test [ Help Index ]

To Publish the test is to assign a particular Test pattern to the respective Category of the Post so that the test is ready for the User.

The Publish Test Utility in the Exam Management allows you to publish the tests which are already created. In publish test you can set the Test Category of the respective test to the format of the Test. This in turn allows you to customize your test settings and helps you to adjust your exams with different Test pattern for the tests of different Categories. As soon as the test is published they are available for the Users and they can give that particular test.

In the publish test page you have got to select the post from the selection box and press the key 'Go' in order to display the Categories ie the Subjects of which the test is to be given in order to qualify for the particular Post. Below are the names of the Categories present in the Post with the respective Tests which are published. One can select the appropriate Test from the selection box and press the 'Publish' button to Publish the tests.

This Utility is only available for those Administrators who have the authority to Manage Exam. See the Accessibility Options for more details.

Warning : It is essential that the Administrator should first Shut the User Server before he does any changes in publish test. Also it is important to unpublish the test before it is deleted or modified as it directly affects the candidates who are giving the test.
