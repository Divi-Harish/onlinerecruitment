Test [ Help Index ]

A Test is a customized pattern where the Administrator can decide the pattern of the test.

The test pattern includes the following :

 Test Name This is the name of the test and must be unique. It is based on the conventions of Name Field and should not exceed more than 20 characters. 
 Description  This is the description of the test of general information. The total characters should not exceed more than 500.
 Question  types  The type of questions ie Multiple Choice or Multiple Answer or True or False Or All Mixed that will be asked in the test.
 Hide Timer  With this option set the Timer clock will not be shown when the test is carried out.
 Time limit  The time limit for the Test. This should not exceed more than 180 minutes ie. 3 hours.
 Total  Questions  These are the total questions for the test which should not exceed 1000.
 Scrolling  With this option enabled, the user can scroll back to the previous question. Disabling it prevents reaching the previous question.
 Min Passing  Score  This option decides the minimum score required for passing the test. Failing to attain this score the candidate will fail     in the test. The minimum passing score can be anything between 0 to 100.
 Negative  Marking  Enabling this option will trigger the negative marking scheme. In the negative marking scheme one mark is given for the correct answer but in case the answer is wrong, one mark is deducted.
